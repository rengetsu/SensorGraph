# SensorGraph

Simple graphical user interface create with Java is made to graph data using JAV libraries.

### Used jar files

  * JFreeChart
  * JSerialComm
  * jcommon
  
### Requirements

  * Git
  * JDK 8 or 11 (to build NetBeans)
  * JDK 9 or above (to run NetBeans)
  * MinGW (optional), to build Windows Launchers
